#Marketer
A simple statistical simulator that allows to recreate consumer movement in a supermarket, written for scholastic purposes.

#Installing
You'll need node.js and npm

```git clone git@github.com:ALCC01/marketer.git
cd marketer
[sudo] npm install
npm start```
